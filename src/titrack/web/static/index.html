How to Use TITrack

1. Enable Game Logging

TITrack reads the game's log file to track your loot. To enable logging:

  1. Open Torchlight Infinite
  2. Go to Settings
  3. Find and click the "Enable Log" button

Without this enabled, TITrack cannot capture any data.

2. Character Detection

TITrack automatically detects your character from the game log:

  • If you've played before with logging enabled, your character is detected automatically on startup
  • If this is your first time, log in with your character and TITrack will detect it
  • To switch characters, just log out and back in (do NOT close the game)

TITrack tracks data separately for each character and league.

3. Sync Your Inventory

To populate the inventory display with your current items:

  1. Open your inventory in-game
  2. Click the Sort button (auto-arranges items)
  3. Repeat for each tab you want to track (Skill, Commodity, Others)

The Gear tab is not tracked because gear prices depend heavily on specific affixes.

4. Automatic Price Learning

TITrack learns item prices automatically when you use the in-game Exchange:

  1. Open the Exchange in-game
  2. Search for any item you want to price
  3. TITrack captures the listed prices and calculates a reference price (10th percentile)

Prices are saved locally and used to calculate your loot value and net worth.

5. Tracking Runs

Once logging is enabled, TITrack automatically:

  • Detects when you enter and leave maps
  • Tracks all items picked up during each run
  • Calculates value based on learned prices
  • Shows your FE/Hour based on active run time only

6. Cloud Sync (Optional)

Share and receive community pricing data by enabling Cloud Sync:

  1. Click the Cloud Sync toggle in the header
  2. When enabled, your Exchange searches are anonymously shared
  3. You'll receive aggregated prices from other users

Benefits:

  • Get prices for items you haven't searched yet
  • See price trends with sparkline charts
  • Help the community by contributing your data

Cloud Sync is completely optional and anonymous. No account required. Only Exchange prices are shared, never manual edits. Data requires 3+ contributors before being shown.

How character detection works

TITrack reads your character name and season from the game log, but Torchlight only writes those details when its Enable Log setting is on at the exact moment the game's login-time data packet arrives. Enabling the log after you're already logged in is not enough — you need to trigger a new login while the log is on.

Try this first

  1. Open Torchlight Infinite and log in to your character as usual.
  2. Open the in-game Settings menu and turn on Enable Log.
  3. Exit all the way back to the Select Character screen (you don't need to quit the client).
  4. Click your character to log back in.

This works because the server re-sends your full character data on every login, and Torchlight only writes it to the log when Enable Log is already on when that packet arrives.

If that didn't work

On some sessions the game records only a truncated placeholder (+player+...) for your character even with logging on. Resetting the client usually clears this:

  1. Fully close Torchlight Infinite (quit to desktop — not just log out).
  2. Reopen the client and log in to your character.
  3. Open Settings and turn on Enable Log.
  4. Exit to Select Character and log back in.

A sort of your inventory (click the Sort button in your bag) also forces a fresh inventory dump, which is a useful extra nudge if detection still doesn't fire.

Other things to check

  • Wrong log file. If you have both the Steam and standalone Torchlight client installed, TITrack may be watching the log for the client you aren't currently playing on. The diagnostic above will suggest a switch when it detects a newer log elsewhere.
  • Game closed. Torchlight only writes to its log while it's running. If you quit the game, the log stops updating and TITrack will flag it as stale.
  • Enable Log doesn't persist. Torchlight resets this toggle to off every time the client restarts. You need to turn it back on each session.
